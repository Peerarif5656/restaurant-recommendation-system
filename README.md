# Restaurant Recommendation System

A machine learning project that builds a personalised restaurant recommendation system using collaborative filtering and similarity-based ranking. The system analyses historical user–restaurant interaction data to recommend restaurants that similar users have rated highly.

---

## Overview

Recommendation systems are widely used in digital marketplaces and food delivery platforms to improve user experience and increase engagement.  
This project demonstrates how machine learning can analyse user behaviour and generate personalised restaurant suggestions.

The model learns from user interaction data and predicts which restaurants a user is likely to prefer based on the preferences of similar users.

---

## Key Features

- Collaborative filtering recommendation model
- User–item interaction matrix
- Cosine similarity for identifying similar users
- Personalised top-K restaurant recommendations
- Ranking evaluation using Precision@K and Recall@K
- Modular Python project structure
